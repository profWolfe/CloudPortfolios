LauncherY Features

LauncherY comes with a lot features, here they are:

  • Powered with jQuery
  • Built on HTML5 Boilerplate
  • Fully responsive. Looks great even on devices with 320px in width
  • Ajax Contact Form with validation
  • Integrated Google Maps
  • Integrated Twitter Feed
  • Awesome animation without Flash
  • Comes with amazing jQuery mOover slider
  • Easy to use Popup Engine. It allows you to call popup windows with any HTML content inside.
  • Dynamic content section with amazing slide animation.
  • Comes with 6 color variations
  • Great browser compatibility (works in IE7 and higher)
  • Validates with HTML5 Doctype
  • Easy to redesign, has very simple and clear API
  • Provided with full-illustrated PDF documentation and layered PSD

Browser Compatibility

LauncherY is tested in the following browsers and compatible with:

  • Internet Explorer 7+
  • Google Chrome 5+
  • Safari 4+
  • FireFox 3.6+
  • Opera 10+
  • iOS 4+ (iPhone, iPod Touch, iPad)
  • Android 2.1+
