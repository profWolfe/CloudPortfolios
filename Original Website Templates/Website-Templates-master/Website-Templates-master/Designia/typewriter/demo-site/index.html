What is TypeWriter?

TypeWriter is a premium JavaScript which allows you to print any phrases. TypeWriter is created by iDangero.us team. It can be used in any purpose, such as welcome message, FAQ, conversation, advertisement, AutoFill etc. (look at the examples below).

This is a standalone script, and therefore it does not require any JavaScript library such as jQuery or MooTools.

It has very simple and clear API. Full illustrated PDF documentation is provided.

TypeWriter Features

  1. Customizable characters delay
  2. Extra delay for individual characters
  3. Custom CSS styles or additional classes for individual characters or phrases
  4. Backspace (character remove) emulation
  5. Support of some control character such as Line Feed (\n) and Horizontal Tabulation (\t)
  6. Small size: 2.52KB minified and 852 bytes minified and gzipped!
  7. Compatiable with Internet Explorer 6 and higher
