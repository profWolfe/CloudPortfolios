LauncherX Features

Here are LauncherX features:

  • Powered with jQuery
  • Ajax Contact Form with validation
  • Awesome animation without Flash
  • Easy to use Popup Engine. It allows you to call popup windows (with a required size) with any HTML content inside
  • Comes with 3 color themes and 2 clock skins
  • Great browser compatibility (works in IE6 and higher)
  • Validates with HTML5 Doctype
  • Easy to redesign, has very simple and clear API
