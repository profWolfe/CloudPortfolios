AWS Cloud S3 Website Portfolio

  • Wanting to further my knowledge in AWS Cloud services like S3, CloudWatch, and Route53 and learn new skills I fully developed and launched a website created utilizing HTML, CSS, and Java-Script.

  • Needing a database to store and publish respective index.html, css, and java-script files I choose S3 a simple object storage service, learning how to host a simple static website.

  • Monitoring metrics for the website brought me to use CloudWatch, I used tags to organize files easily, these metrics gave insight on how the website is doing.

  • On previous websites published in the same fashion I wanted a custom domain name, and used Route53's DNS service to choose a custom name then CloudFront to connect the S3 files to the domain name, publishing the websites globally with a .com custom domain name.

  • All respective files are updated and maintained in GitHub through Git.
